# English Verb Tenses Cheat Sheet in LaTeX

## Description

This repository contains a LaTeX document designed as a comprehensive cheat sheet for English verb tenses. The document includes explanations, structures, and examples for commonly used verb tenses in English, including:
- Present Simple
- Past Simple
- Present Perfect
- Present Continuous
- Past Continuous
- Future Simple
- Future Continuous
- Past Perfect

The cheat sheet provides:
- Detailed explanations for each tense
- Affirmative, negative, and question forms
- Guidelines on when to use `had`, `have`, and `has`

## Contents

- `verb_tenses_cheat_sheet.tex`: The main LaTeX file with the formatted document on English verb tenses.

## Usage

To compile the document:
1. Clone the repository.
2. Open the `.tex` file in a LaTeX editor (such as Overleaf or TeXShop).
3. Compile to generate a PDF file.

This resource is useful for students, educators, and anyone looking to have a quick reference for English verb tense usage.
